# Electrical Cat / Naoki Takahashi Portfolio

This is the source code for my personal portfolio website, [nya3neko2.dev](https://nya3neko2.dev).
It showcases my projects, research, and technical journey in Software Engineering and Mathematics.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Neo-Brutalism Design (CSS)
- **Deployment**: Cloudflare Pages
- **Features**:
  - Multi-language support (JA/EN).
  - Tweet archive visualization and analytics using parsed CSV data.
  - Interactive project moduli and custom components.

## Local Development

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Dev Server**
   ```bash
   npm run dev
   ```

3. **Build**
   ```bash
   npm run build
   ```
